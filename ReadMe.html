xWhois v1.0

This is a simple mod and block combo that allows you to add Whois search capabilities to your site.

There are 2 types of blocks with html templates. The first block looks best as a side, the second block looks best as a center. Choose the style you like!

Installation

Copy the package to your Xoops modules directory and install as any other module. There are no databases. This is a very simple module and block php code section.
